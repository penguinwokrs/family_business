# 法人化プロジェクト詳細設計書 (Incorporation Project Master Plan)

更新日: 2025年12月29日
作成者: 子（資産管理会社 代表予定）

本資料は、お母様の所有する不動産（賃貸物件）の法人化による資産承継、税務最適化、および将来の事業拡大に向けた詳細計画をまとめたものです。

---

## 1. プロジェクトの背景と目的

### 現状の課題 (Persona & Pain Points)

* **お母様（被承継者）**:
  * 介護職（年収120万）と不動産収入（年660万）があるが、所得税・住民税・国保等の負担が重く（約230万円/年）、手取りが圧迫されている。
  * 現金が蓄積され続けることで、将来の相続税負担が増加する懸念がある。
* **貴方（承継者）**:
  * 自身の事業（フリーランス）に加え、将来的に「大家業の拡大」を志向している。
  * 法人化を通じてキャッシュフローを改善し、次の物件購入のための資金（内部留保）を作りたい。

### 目的 (Goals)

1. **お母様の手取り確保**: 税金を削減し、生活に必要な資金（年300万円強）を確実に、かつ非課税に近い形で渡す。
2. **資産移転と節税**: 所得分散により世帯トータルの税負担を劇的に下げ、その分を次世代（法人）に残す。
3. **事業拡大の基盤**: 法人に資金をプールし、5年後を目処に大規模修繕や新規物件取得を行う。

## 2. 法人化スキームの概要 (Scheme Overview)

「建物のみ譲渡」方式を採用し、お母様の土地にお金を払って借りる形をとります。

* **法人形態**: **合同会社 (GK)**
  * 設立コストが安い（約15万円前後）ため、家族経営に最適。
* **物件の取扱い**:
  * **土地**: お母様所有のまま。法人は「地代」を支払う（借地）。
  * **建物**: 法人がお母様から買い取る（売買）。
* **評価額と対価**:
  * **建物価格**: **1,000万円**（固定資産税評価額相当と仮定）
  * **支払方法**: 法人からお母様へ、**年額220万円 × 約4.5年** の分割払いで支払う（未払金処理）。

### ※スキームの適法性と安全性

本計画では、設立時に1,000万円の資本金を用意して支払う「一括決済」ではなく、**「分割払い（未払金計上）」**を採用します。

* **見せ金リスクの回避**: 設立直後に資本金全額を引き出して支払う行為は「見せ金（資本金の仮装）」とみなされるリスクがありますが、分割払いならそのリスクがありません。
* **資金調達不要**: 最初に現金を用意する必要がなく、法人の家賃収入から無理なく支払えます。
* **非課税枠の活用**: 分割受取により、お母様は毎年安定した「非課税キャッシュ」を生活費として受領できます。

## 3. 収支シミュレーションと税務メリット

最も効果が高い「最適化プラン」における数値です。

### A. 年間キャッシュフロー比較 (お母様の手元資金)

| 項目 | 現状（個人事業） | **法人化後（最適化案）** | 備考 |
| :--- | :--- | :--- | :--- |
| **給与収入** | 120万円 | 120万円 | 介護職（継続） |
| **役員報酬** | 0円 | **48万円** (月4万) | 給与所得控除内で**非課税** |
| **不動産収入** | 660万円 | **40万円** (地代) | 固定資産税相当額に設定 |
| **建物売却益** | 0円 | **100万円** (分割返済) | **非課税** (元本の受取) |
| **(収入合計)** | **780万円** | **308万円** | |
| **税金・社保** | ▲ 約228万円 | **▲ 約10万円** | 所得が激減するため最低水準へ |
| **手取り合計** | **約 552万円** | **約 298万円** | ※生活費(300万)にほぼ到達 |

* **ポイント**: お母様の手取りは減りますが、生活費は十分確保され、かつ「税金」として消えていたお金がなくなります。差額はすべて「法人」に残ります。
* **オプション**: それでも足りない場合は、お父様を役員に追加し、さらに同額(月4万円)を支給することで、世帯収入を無税で増やすことが可能です。

### B. "世帯全体"での節効果

| 項目 | 効果金額 | 内容 |
| :--- | :--- | :--- |
| **所得税・住民税** | **約 ▲160万円/年** | 累進課税(最大33%等)から法人税等(均等割7万)へのシフト |
| **社会保険料** | **約 ▲50万円/年** | お母様の国保・介護保険料が最低ランクへ低下 |
| **単年メリット** | **合計 約 211万円** | これが毎年「法人」にプールされます |

※別途、法人の維持コスト（税理士報酬・均等割等で年約50万円）がかかりますが、それを差し引いても **年間 約150〜160万円の純プラス** となります。

### C. お母様の個人事業収入（120万円）について

介護職としての収入120万円は、青色申告特別控除（最大65万円）や基礎控除（48万円）を適用することで、課税所得をほぼゼロにできます。
これにより、この収入に対する**所得税・住民税は実質「無税（ゼロ）」**に抑えられ、手取り額＝収入額として確保可能です。

## 4. 法人の収支と将来戦略 (Business Roadmap)

法人に残ったお金をどう使うか（貴方の戦略）についてです。

### Step 1: 内部留保の蓄積 (1〜5年目)

建物代金（分割）の支払期間中は、資金繰りを安定させつつ実績を作ります。

* **法人収入**: 家賃 660万円
* **法人支出**:
  * 地代: 40万円
  * 建物分割払: 100万円（※キャッシュアウト）
  * 経費・税金: 約60万円
  * お母様報酬: 48万円
* **手残り現金**: 約 412万円/年
  * ※貴方（子）への役員報酬は **0円** とし、**「法人の預金（軍資金）」を最大化** することを推奨します（将来の融資対策）。

### Step 2: 事業拡大と合流 (5年目以降：個人事業最適化フェーズ)

建物代金の支払いが完了し、減価償却も終わるタイミングで、貴方の個人事業を法人に統合します。
このフェーズでは、**「役員報酬 550万円」**に設定するプランが最も経済合理的です。

* **報酬設定**: 年550万円（月額46万円）
  * **法人収支**: ほぼトントン（赤字にならず、銀行融資に有利な決算書を維持）
  * **税金・社保**: 最適化された負担率。
* **NISA戦略**:
  * 手取り月額 約36万円のうち、生活費11万円を除いた**25万円**を新NISAに投資。
  * **最短6年**で生涯投資枠（1,800万円）を埋め切ることが可能。
* **目的**: 単なる節税だけでなく、法人を活用した**「個人の資産形成（FIRE準備）」**を加速させます。

## 6. 今後のアクションプラン

1. **お母様への最終説明**
    * 別添資料『お母様向け説明資料_法人化メリット』を使用し、手取りの変化（生活に不安がないこと）を説明。
2. **定款作成と法人設立**

    * 評価額証明書を取得し、売買価格（1,000万円）と分割払い条件を確定。
4. **税務署への届出**
    * 「無償返還の届出書」（地代を安くするための必須書類）の提出。

## 参考資料：税務最適化ロジック (Appendix)

### 所得税の「195万円の壁」について

所得税率は「課税所得」が195万円を超えると5%から10%に倍増します。

* **最適ライン**: 額面年収 330〜350万円
* **理由**: 各種控除（給与控除、基礎控除、社会保険料控除、小規模企業共済等）を引くと、課税所得が195万円以下に収まり、**実質負担率約5%**で済みます。
* 本プロジェクトの「Phase 2（貴方の報酬設定）」や「お母様の別収入調整」では、常にこの効率的なラインを意識して設計されています。
