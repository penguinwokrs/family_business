# 法人化（建物譲渡）詳細ケーススタディ (Incorporation Case Study: Building Transfer)

調査日: 2025年12月27日

## 1. モデルケースの概要

* **主体**: 新設する資産管理法人（合同会社を想定）
* **地代**: **固定資産税と同額**（約30〜40万円/年）
  * 土地は移転せず、お母様から法人が借りる（地代を払う）形をとります。
  * **理由**: お母様の不動産所得をゼロにし、税・社保負担を増やさないため。

---

## 2. 収支シミュレーション (Cash Flow Simulation)

**【前提数値】**

* **年間家賃収入**: **660万円** (55万円×12ヶ月) - 満室想定
* **建物買収価格**: **1,000万円** (仮定: 固定資産税評価額相当で売買)
  * ※法人はこの1,000万円をお母様に支払う（未払金として分割払いも可）。

### A. 支出の部（法人から出るお金）

| 項目 | 金額(年額) | 計算根拠・備考 |
| :--- | :--- | :--- |
| **(1) 土地の地代** | **約 75〜100万円** | お母様へ支払う。「無償返還の届出」を提出し、固定資産税の2〜3倍程度を設定。 |
| **(2) 建物減価償却** | **約 200〜300万円** | 中古木造/鉄骨（耐用年数超過）の場合、4年程度で償却。初期数年は大きな赤字（節税）を作れます。 |
| **(3) 固定資産税** | **約 50万円** | 建物分（所有者として）、土地分（地代に含まれるが実質負担）。 |
| **(4) その他経費** | **約 50万円** | 税理士報酬、修繕費積立、火災保険など。 |
| **小計(経費)** | **約 375〜450万円** | (償却期間終了後は(2)がなくなり、利益が急増します) |

### B. 利益と分配（アセットアロケーション）

**償却期間中の法人利益（償却前）**: 660万 - (150万:地代+固定税+経費) = **約 510万円** (キャッシュフローベース)
これらをどう配分するかで、「資産増加」か「節税」かが決まります。

#### パターン1: 「資産移転」重視（貴方への給与最大化）

貴方（子）に給与を払い、法人の利益と現金を貴方に移します。

* **役員報酬（子）**: **月額 30万円 (年360万円)**
* **役員報酬（親）**: **月額 5万円 (年60万円)** ※年金カットされない範囲、生活費足し
* **法人利益**: 510万 - 420万 = **90万円** (減価償却を入れると帳簿上は大赤字→法人税均等割のみ7万円)
* **地代**: 固定資産税と同額（約30〜40万円）
* **法人税等の合計**: **約 7万円** (赤字でもかかる均等割7万円のみ)
* **消費税**: 免税（2年前まで）
* **社会保険料**: **0円** (役員報酬1万円のため未加入)
* **法人の合計負担**: **約 7万円** (減価償却を入れると帳簿上は大赤字→法人税均等割のみ7万円)
* **効果**:
  * お母様の資産増加額: 地代100万 + 給与60万 = **160万円/年** (ここから税金・生活費で消える計算)
  * 貴方の資産増加額: **360万円/年** (給与所得控除後の税金は安い)
  * **判定**: 最もオーソドックスで推奨される形です。

#### パターン2: 「内部留保」重視（将来の大規模修繕へ）

役員報酬を抑え、法人に現金を貯めます。

* **役員報酬（子）**: **月額 10万円 (年120万円)**
* **役員報酬（親）**: **なし**
* **法人利益**: 510万 - 120万 = **390万円**
* **【個人の負担（お母様）】**
別給与(120万)＋地代(固定資産税同額)＋役員報酬(12万) - 共済控除(84万) ＝ 課税所得ゼロ（または最低ライン）

* **所得税・住民税**: **約 5〜10万円** (別給与に対する税額のみ)
* **個人事業税**: **0円** (不動産所得ゼロのため)
* **国保・年金**: **別給与天引き分のみ** (負担増なし)
* **効果**:
  * 法人口座に毎年約300万円以上の現金が貯まります（税引後）。
  * 5〜10年後の建て替えや大規模修繕の頭金になります。
  * **判定**: 大規模修繕が近い（5年以内）ならこちら。

#### パターン3: 「事業拡大」重視（内部留保で再投資）

貴方の「大家業拡大」という目的に特化したプランです。
役員報酬を生活費の必要最低限に抑え、法人にキャッシュを全力で貯めて、次なる物件購入の頭金にします。

* **戦略**:
  * **フェーズ1（1-5年目）**: 役員報酬を低く設定（例: 月10万円など）し、毎年400万円以上の現金を法人に留保する。
  * **フェーズ2（5年目以降）**: 貯まった資金（約2,000万円）を頭金に、融資を引いて **5,000万円〜1億円規模の収益物件（一棟マンション等）** を購入する。
  * **フェーズ3（拡大期）**: 2棟目からの家賃収入でさらにキャッシュフローを加速させる。

* **メリット**:
  * **融資実績**: 法人として黒字決算を続けることで、金融機関からの信用（トラックレコード）が作れます。
  * **再投資効率**: 個人の給与として受け取ると税金・社保で30%引かれますが、法人なら実効税率約22〜25%（小規模法人）で済み、再投資への効率が最大になります。
  * **建替・土地購入**: 将来、現在の土地の隣地購入や、建替費用としても使えます。

* **判定**: **「将来、事業を拡大したい」「資産家になりたい」** という強い意志がある場合は、このプランが最適解です。

## 3. 親（お母様）のキャッシュフロー変化

法人化するとお母様の収入構造がガラリと変わります。

| 項目 | 現状（個人） | 法人化後 | 備考 |
| :--- | :--- | :--- | :--- |
| **収入** | 家賃 660万円 | 地代 100万円<br>役員報酬 60万円<br>売買代金 1,000万円(※分割受取) | 売買代金は「元本」なので非課税。給与と地代のみ課税。 |
| **手残り** | 多すぎ（資産増） | **適正（生活費相当）** | 余分な蓄財を防ぎ、相続財産を増やさない。 |

※**売買代金1,000万円の扱い**:
法人はお母様に1,000万円を支払う義務があります。これを銀行から借りて一括で払うか、あるいは「法人からお母様への借金」として、毎年100万円ずつ10年かけて返すことで、お母様の老後資金にします。

## 4. 総括（推奨プラン）

**「パターン1（資産移転重視）」** での運用をお勧めします。

1. **地代（約100万円）**をお母様に払う。（これは必須）
2. **貴方への役員報酬（年300〜400万円）**を設定し、家賃の過半数を貴方に移す。
3. **減価償却（4年間）**を活用し、その間は法人税を最低限（7万円）に抑える。
4. 償却が終わる5年後以降は、役員報酬を増やすか、修繕積立をして利益を圧縮する。

このスキームにより、**「お母様の資産凍結リスク回避（認知症対策）」**と**「年間300万円以上の次世代への資産移転」**が同時に達成できます。
