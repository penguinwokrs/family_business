# 法人化シミュレーション：新NISA攻略・収支均衡プラン (Balanced Aggressive)

作成日: 2025年12月28日
更新：報酬550万・収支トントン・NISA最大化

本プランは、役員報酬を**550万円**に設定し、**会社の利益をほぼプラマイゼロ（トントン）**に保ちつつ、個人の手取りを最大化してNISAに資金を流す「バランス型・資産形成プラン」です。

## 1. 収支設計 (年商800万：バランス最適化)

会社の赤字を出さず、かつ法人税も均等割（7万円）のみに抑える、最も美しい着地点です。

| 項目 | 設定金額 | 詳細・計算根拠 |
| :--- | :--- | :--- |
| **A. 法人売上** | **8,000,000円** | |
| **B. 役員報酬** | **年額 5,500,000円** (月46万) | **※変更点**: 600万から50万下げ、会社の赤字を解消。 |
| **C. 社会保険料(会社負担)** | **約 800,000円** | 報酬額面×約14.5%で試算。 |
| **D. 社宅家賃経費** | **年額 1,200,000円** | 月10万の会社負担（維持）。 |
| **E. 旅費日当** | **年額 360,000円** | 月3万の非課税支給（維持）。 |
| **F. その他経費** | **年額 400,000円** | 通信費、消耗品、税理士報酬など。 |
| **G. 倒産防止共済** | **0円** | 廃止。資金を個人へ。 |
| **H. 法人損益** | **▲ 260,000円** (ほぼ0) | **事実上のトントン**。少額の赤字は繰越欠損金として処理可能で、税務上も健全。 |

## 2. 個人の手取りとNISA後の「手残り」

役員報酬550万円における、リアルな生活費シミュレーションです。

### 月額キャッシュフロー (手取り)

* **額面給与**: 458,333円 (年550万)
* **控除 (社保・税金)**: ▲ 約105,000円
  * 社会保険料(個人): 約68,000円
  * 所得税・住民税: 約37,000円 (概算)
* **社宅自己負担**: ▲ 20,000円 (天引き)
* **給与手取り**: **約 333,000円**
* **(+) 旅費日当**: **30,000円**
* **(=) 月間可処分キャッシュ**: **363,000円**

### NISA投資と「手残り生活費」

NISAに「毎月25万円」突っ込むアグレッシブな投資を行った場合の、残りの生活費です。

| 項目 | 金額 (月額) | 備考 |
| :--- | :--- | :--- |
| **① 可処分キャッシュ** | **363,000円** | 家賃支払い済みの状態 |
| **② NISA投資** | **▲ 250,000円** | **年額300万円**ペース (6年で満額) |
| **③ 手残り生活費** | **113,000円** | **食費・水道光熱費・通信費・娯楽費** |

### 生活費 11.3万円の内訳イメージ

家賃は既に会社払いで解決済みなので、11.3万円あれば普通に生活可能です。

* 食費: 50,000円
* 水道光熱費: 15,000円
* 通信費: 5,000円
* 日用品・娯楽: 43,000円
* **合計: 113,000円** → **生活可能**

### 結論

**「報酬550万円プラン」**であれば、会社の決算書を綺麗に保ちつつ（銀行融資が必要になった時も有利）、**6年でNISA生涯枠1,800万円を埋め切る**ことが可能です。

家賃を除いて**月11.3万円**で生活できるなら、このプランが最も経済合理的かつ持続可能性が高い選択肢となります。
