# 法人化（建物譲渡）詳細ケーススタディ：親の節税最大化Ver. (Maximum Tax Saving for Parent)

調査日: 2025年12月28日

## 1. シミュレーションの方針変更

* **変更点**: 貴方（子）への報酬は一旦考えず、**「お母様（親）の節税と手取り最大化」**に振り切った場合のシミュレーションを行います。
* **目的**: お母様の税金（所得税・住民税）を極限まで減らしつつ、将来の相続税も増やさない最適なバランスを探ります。

---

## 2. 親（お母様）の収入源と税務リスク管理

別収入（120万/年）と地代（100万/年）があるため、役員報酬は社会保険料への影響を最小限に抑える額に設定します。

### A. 地代収入（必須）

* **金額**: **年額 100万円** (月額 約8.3万円)
* **根拠**: 固定資産税の約3倍程度。「無償返還の届出」を提出し、借地権課税を回避しつつ、高すぎない設定にします。
* **税金**: 不動産所得として課税されますが、少額のため小規模企業共済で相殺します。

### B. 役員報酬（社会保険加入ライン）

* **金額**: **月額 5,000円 〜 1万円** (年額 6〜12万円)
* **社会保険**: 報酬が低すぎるため、年金事務所の判断によっては**社会保険に加入できません**（国民健康保険・国民年金のまま）。
  * **理由**: 月額6.3万円未満は標準報酬月額の最低等級を下回ります。年金事務所は「社会保険料も払えない報酬で加入するのはおかしい」と判断することがあるためです。
  * **対策**: あえて「加入しない」ことを選び、会社の社会保険料負担をゼロにする戦略です。
* **勤務実態**: 非常勤役員として、月1回の会議出席やアドバイス程度で十分説明がつきます。

### C. 建物売却代金（非課税キャッシュ）

* **金額**: **1,000万円** (建物の対価)
* **受取方法**: **分割払い (年額 100万円 × 10年)**
* **メリット**: これは「資産の売却代金」なので**所得税も住民税も一切かかりません**。
* **効果**: 毎年100万円の**「完全非課税の手取り」**がお母様に入ります。税金を払わずに生活費を確保する最強の手段です。

---

## 3. 親（お母様）の「節税」最大化スキーム

お母様には**「地代」等の他に、別仕事による給与収入が年120万円**あるため、これを合算して節税策を再構築します。

**【収入の整理】**

1. **別仕事の給与**: 120万円 (給与所得控除55万引くと所得65万)
    * ※別のお仕事で社会保険（または国保）に加入済みと想定。
2. **地代収入**: 100万円 (不動産所得)
3. **役員報酬**: 12万円 (少額なため影響ほぼなし)

* **合計所得見込**: 約165万円前後 (控除前)

### (1) 小規模企業共済への加入 (必須)

* **概要**: 個人事業主（または会社役員）の退職金積立制度。
* **掛金**: **月額 7万円 (年額 84万円)** ※最大額
* **効果**: 掛金全額が「所得控除」になります。
  * 合算所得(約165万) - 共済控除(84万) - 基礎控除(48万) = **課税所得 約33万円**
  * **結果**: 所得税率5%区分のまま。税金は年間数万円程度に抑え込めます。
  * これが無いと、所得165万に対してまともに課税され、税負担が重くなります。
* **加入資格**: 会社役員になれば高齢でも加入可能です（共同経営者としての加入も検討可）。

### (2) 経営セーフティ共済（倒産防止共済）

* **概要**: 連鎖倒産防止のための積立。
* **掛金**: **月額 20万円 (年額 240万円)** ※最大額、法人が加入
* **活用**: これは法人の節税策ですが、法人の利益を圧縮し、将来お母様が退職する際の「退職金原資」として溜めておけます。

---

## 4. 収支シミュレーション結果（親の手取り）

別仕事の収入(120万)と極低役員報酬を加味した再計算結果です。

| 項目 | 法人化前（現在） | 法人化後（対策済） |
| :--- | :--- | :--- |
| **別仕事収入** | 120万円 | 120万円 |
| **不動産収入** | 660万円 | 地代 100万円 |
| **役員報酬** | 0円 | 6〜12万円 |
| **売却代金(非課税)** | 0円 | **100万円** |
| **合計現金流入** | **780万円** | **約 330万円** |
| **税金・社保** | **約 200万円強** (所得税高額) | **約 10万円** (激減) |
| **実質手取り** | **約 580万円** (資産過多) | **約 320万円** (適正) |
| **資産増加** | 毎年500万以上増える | 生活費で消費可能 |

※対策後の税金は、小規模企業共済で圧縮し、かつ社会保険料（国保・国民年金）は別仕事の方で払っているか、個人の負担額のみになる想定です。

### 結論

現在の別仕事（年120万）で社会保険または国民健康保険を対応し、新法人の報酬は**月額1万円**に抑えるのが最適です。
