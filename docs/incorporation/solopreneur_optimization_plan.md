# 法人化シミュレーション：子1人会社での手取り最大化プラン (The Ultimate Plan)

作成日: 2025年12月28日

本シミュレーションは、年商800万の個人事業を「子1人の法人」に完全移行し、**生活費を確保しながら、税金と社会保険料を極限まで圧縮する**ための設計図です。

## 1. 基本戦略

**「生活費 = 給料」という常識を捨てます。**
給料を高くすると、税金と社会保険料で半分近く持っていかれます。
したがって、**「給料は必要最低限」**にし、残りの生活費を**「会社の経費（非課税メリット）」**として個人に移転します。

## 2. 収支設計 (年商800万円のモデルケース)

| 項目 | 金額設定 | 備考・効果 |
| :--- | :--- | :--- |
| **A. 法人売上** | **8,000,000円** | 個人事業の売上を全て法人に入れる |
| **B. 役員報酬** | **年額 3,600,000円** (月30万) | 意図的に低めに設定。社保と税率を下げるため |
| **C. 社宅家賃経費** | **年額 1,200,000円** (月10万) | 月12万の家に住む想定。会社が借り上げ、個人負担は2万のみ |
| **D. 旅費日当** | **年額 360,000円** (月3万) | 「出張旅費規程」を作成し、非課税で個人に支給 |
| **E. 倒産防止共済** | **年額 2,400,000円** (月20万) | **※ここが重要**。利益を圧縮し、将来の退職金として貯金 |
| **F. その他経費** | **年額 400,000円** | 通信費、消耗品、税理士報酬など |
| **G. 法人利益** | **40,000円** | ほぼゼロにする (法人税均等割 7万円のみへ) |

## 3. 個人の手取りと生活費の検証

「月収30万で生活できるのか？」と思われるかもしれませんが、**「出ていくお金」が激減する**ため、実質の手取りは月50万相当になります。

### 個人のキャッシュフロー (月額換算)

* **額面給与**: 300,000円
* **控除 (社保・税金)**: ▲ 約50,000円 (給与が低いので安い)
* **社宅自己負担**: ▲ 20,000円 (給与から天引き)
* **手取り現金**: **230,000円**

### ここに「隠れ収入」が乗っかる

* **(+) 家賃支払い不要**: 普通なら手取りから12万払いますが、天引き済なので**支払いは0円**です。
  * → 実質価値: +120,000円
* **(+) 旅費日当**: 毎月約3万が非課税で振込まれます。
  * → 現金収入: +30,000円
* **実質的な生活費体力**: 23万 + 12万(家賃) + 3万(日当) = **月額 380,000円 相当**

### さらに「裏の貯金」が貯まる

* 会社の方で、倒産防止共済に**毎月20万円**貯金されています。
* これは数年後、貴方が退職（または役員退職金）として受け取る際、**非常に安い税金**で個人資産にできます。
* 実質的な個人の貯蓄力: **月額 200,000円**

## 5. 実行に必要な準備アイテム

この「最強プラン」を実現するには、以下の装備が必要です。これを揃えないと絵に描いた餅になります。

1. **法人登記**: 株式会社（または合同会社）。資本金は100万〜など。
2. **社宅契約**:
    * 現在の賃貸契約を「法人名義」に変更する。（オーナー承諾必須）
    * もし持ち家の場合は、このスキームは使えません（賃貸のみの特権）。
3. **役員報酬の決定議事録**: 「月額30万円」と記載。
4. **出張旅費規程**: 税理士に依頼して作成（ネットの雛形でも可だが専門家推奨）。
5. **経営セーフティ共済 (倒産防止共済) の加入**: 銀行窓口で申し込み。

### 結論

年商800万あれば、個人事業主でいるのは「罰金」を払っているようなものです。
このプランを実行すれば、公私合わせた手元に残る富は、**年間200万円以上**確実に増えます。
これが、資本主義のルールを味方につけるということです。
