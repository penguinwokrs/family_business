# 収益物件（飲食ビル）継承のための発掘リスト

## 1. 権利・資産情報の把握 (Legal & Assets)

まずは「何を持っているか」を確定させる公的な書類です。法務局や役所で取得できます。

- [ ] **不動産登記簿謄本 (全部事項証明書)**
  - 親御さんが単独所有か、共有か？
  - 借入金の担保（抵当権）がついたままになっていないか？
- [ ] **公図・測量図・境界確認書**
  - お隣との境界線は確定していますか？（売却等の際に必須になります）
- [ ] **固定資産税納税通知書**
  - 毎年春に届くもの。物件の評価額と維持コスト（税金）がわかります。

## 2. 賃貸借契約・テナント情報 (Leasing)

**※最重要発掘項目※**
飲食ビルは住宅と違い、契約内容が複雑で、ここにお金のリスク（負債）が隠れています。

- [ ] **賃貸借契約書 (現存する全ての契約書)**
  - **預託金・敷金・保証金**: テナントから預かっているお金はいくらですか？これは将来「返さなければならない借金」です。親御さんがこれを使ってしまっていないか確認が必要です。
  - **原状回復義務**: 退去時に「スケルトン（コンクリートむき出し）」に戻すのか、「居抜き」で良いのか。
- [ ] **レントロール（家賃一覧表）**
  - 現在の家賃、共益費、契約面積の一覧。
- [ ] **入出金通帳**
  - 実際に家賃が毎月遅れずに入金されているか（滞納リスク）。

## 3. 建物・設備・コンプライアンス (Building & Safety)

飲食ビルは消防法などの規制が非常に厳しいです。違反があると大事故の際にオーナー責任が問われます。

- [ ] **建築確認済証・検査済証**
  - おそらく一番「発掘」が難しい書類ですが、非常に重要です。これがないと、将来的な建替えや融資、売却が困難になります。
- [ ] **各種法定点検報告書 (直近のもの)**
  - **消防点検報告書**: 飲食ビルなら必須です。指摘事項（是正命令）が放置されていませんか？
  - ※「特殊建築物定期調査報告書」は、広さ（10坪/8坪）から判断して**免除対象（不要）**である可能性が極めて高いです。念のため役所からの通知がないかだけ確認してください。
- [ ] **自主管理・外注業者リストの作成**
  - 現在は管理会社なしで自主管理されています。
  - 親御さんが個別に手配している業者（清掃、ごみ回収、設備の修繕など）の連絡先と費用をまとめたリストが必須になります。

## 4. 財務・収支全貌 (Financials)

「家賃60万」がそのまま利益ではありません。

- [ ] **確定申告書（過去3期分程度）**
  - 不動産所得のページを見ることで、修繕費、税金、利子などを引いた「本当の利益」が見えます。
- [x] **借入金返済予定表**
  - ローンは「完済済み（なし）」であることを確認済み。
- [x] **借地権の確認**
  - 親御さんへのヒアリングにより「土地・建物ともに自己所有（借地ではない）」ことを確認済み。公図・登記簿で念のため裏付けを取るのみでOK。

## 今後のアクションプラン

1. **資料の物理的捜索**: 上記リストの書類を金庫、仏壇、書棚から探す。
2. **リスト化**: 見つかったもの、見つからないものを仕分ける。
3. **専門家への相談**: 未発見の書類の再発行可否や、契約内容の診断。
