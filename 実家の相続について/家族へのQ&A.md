1. # **相続対策Q\&A（家族会議用）**

2. ### **Q1. なぜ「実家」を賃貸住まいの兄弟のどちらかが相続するのですか？**

3. **A.** **「家なき子特例」という最強の節税ルールを使うためです。** 実家（100坪）をそのまま相続すると税金が高くなりますが、持ち家のない兄弟が相続すると、土地の評価額が**80%オフ**になります。お姉様は親名義のマンションに住んでいるため、この特例を使えません。家族全体の納税額を最小（数百万円単位の節税）にするための「戦略的な役割分担」です。

4. ### **Q2. 実家を相続した兄弟が、売却代金を一人占めすることになりませんか？**

5. **A.** **いいえ、遺言書で「換価分割（かんかぶんかつ）」を指定します。** 「書類上の相続人」は一人になりますが、遺言書によって「売却して得た現金を3兄弟で分けること」を義務付けます。これにより、最終的な受取額の公平性を保ちます。

6. ### **Q3. なぜ「今」実家を売ってはいけないのですか？**

7. **A.** **売ってしまうと「80%オフ」の権利が消え、現金（100%課税）になるからです。** 今売ると、売却代金すべてが相続税の対象になります。相続した後に売れば、安い評価額で税金を払った後、市場価格で売却してキャッシュを手にできるため、手元に残るお金が最大化されます。

8. ### **Q4. 賃貸住まいの兄弟が家を買うとどうなりますか？**

9. **A.** **購入から3年経つと、特例（80%オフ）が使えなくなります。** もし相続前にマイホームを購入する予定がある場合は、必ず家族で相談してください。タイミングを数年ずらすだけで、家族全体で支払う税金が数百万円変わる可能性があります。

10. ### **Q5. お姉様が住んでいるマンションはどうなりますか？**

11. **A.** **お姉様がそのまま相続することを推奨します。** 住まいを確保した上で、実家の売却代金から他の兄弟とのバランスを調整する形が最もスムーズです。  

| 項目 | 【パターンA】生前に売却 | 【パターンB】相続後に売却 |
| :--- | :--- | :--- |
| 土地の評価（実家） | 6,000万円（現金として評価） | 1,200万円（80%減額を適用） |
| マンションの評価 | 2,000万円 | 2,000万円 |
| 現金の評価 | 2,000万円 | 2,000万円 |

| 基礎控除額 | 4,800万円 | 4,800万円 |
| 課税される遺産額 | 5,200万円 | 400万円 |
| 相続税の総額（概算） | 約630万円 | 約40万円 |
