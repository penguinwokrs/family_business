# 「未払金（分割払い）」採用時の通帳・お金の流れシミュレーション

作成日: 2025年12月28日

「推奨プラン（建物代金1,000万円を10年分割払い）」を採用した場合の、**通帳の記帳内容（お金の動き）**を時系列で再現しました。

---

## 1. 設立当初（1ヶ月目）：初期設定

まず、会社を作るための資本金だけが動きます。1,000万円という大金は動きません。

### 🅰️ 子（または母）の個人通帳

| 日付 | 摘要 | 出金金額 | 入金金額 | 残高 | メモ |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 4/1 | 資本金払込 | 1,000,000 | | 〇〇〇 | 新設法人の資本金として振込 |

### 🅱️ 法人の通帳（新設）

| 日付 | 摘要 | 出金金額 | 入金金額 | 残高 | メモ |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 4/15 | 資本金預入 | | 1,000,000 | 1,000,000 | 会社がスタート |
| 4/20 | 司法書士報酬 | 250,000 | | 750,000 | 設立費用などの支払 |
| 4/25 | **登録免許税** | 200,000 | | 550,000 | 建物の登記移転費用（※概算） |

**★ポイント**:

* この時点で**「建物売買契約」**は完了していますが、**通帳上は1円も動きません**。
* 契約書に「代金は分割払い」と書いてあるため、法的な所有権だけが会社に移ります。

## 3. 決算・支払い月（年1回）：建物の分割払い

契約で定めた日（例：毎年の決算月や誕生日など）に、約束通り**「1年分の建物代金（分割分）」**を支払います。

## 4. 会計ソフト上の処理（裏側の動き）

通帳には出ませんが、会計上は以下のように処理され、しっかり「借金返済」の記録が残ります。

1. **売買契約日（4/1）**
    * `（借方）建物 1,000万円 / （貸方）未払金 1,000万円`
    * 意味：建物という資産を手に入れたが、お母様への借金（未払金）が1,000万円ある状態。

2. **分割払いの日（毎年3/31）**
    * `（借方）未払金 100万円 / （貸方）現預金 100万円`
    * 意味：お母様への借金を100万円返した。残りの借金は900万円。

## まとめ

* **通帳の見た目**: 設立時にドカンと大金が動くことはありません。家賃がコツコツ入り、年に1回お母様へまとまったお金（分割金）が出ていくだけの、非常にきれいな流れになります。
* **資金繰り**: 最初に1,000万円を用意する必要がなく、入ってきた家賃から払うので、資金ショートの心配もありません。
