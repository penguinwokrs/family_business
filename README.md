# Family Business Succession Project (家族経営事業承継プロジェクト)

本プロジェクトは、所有する不動産賃貸事業の次世代への円滑な承継、および法人成りを活用した税務最適化（節税効果）の分析と実行計画をまとめたものです。

## プロジェクトの目的

1. **相続税対策**: 高齢のお母様への現金蓄積（将来の相続財産増加）を抑制する。
2. **所得税・住民税の削減**: 個人事業主としての高税率状態から、法人税率（実効税率低下）への移行。
3. **事業承継**: 法人設立を通じて、不動産管理事業を次世代（子）へスムーズに移管する。

## 分析結果サマリ (Executive Summary)

### 1. 法人化の効果

シミュレーションの結果、法人成り（建物のみ譲渡）を行うことで、以下のメリットが確認されました。

* **年間節税額**: 約 **211万円** のメリット（所得税・住民税・社会保険料の合計削減額）。
* **10年間の効果**: 約 **2,000万円** 以上の差が生じる試算。
* **コスト回収**: 法人設立・移転コスト（約100万円弱）は初年度で回収可能。

### 2. 採用するスキーム（建物譲渡モデル）

* **売買対象**: 建物のみ（土地はお母様から法人が借りる）。
* **売却価格**: **1,000万円**
  * *根拠*: 固定資産税評価額を0.7で割り戻した時価相当額（税務否認リスク低減のため）。
* **決済方法**: **未払金（金銭消費貸借契約）**
  * 銀行融資を受けず、法人がお母様へ10年間かけて分割返済する。
  * **金利**: **無利息**（役員から法人への貸付のため税務上問題なし）。
  * **返済額**: 年間100万円（法人のキャッシュフロー範囲内）。
