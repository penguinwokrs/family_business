# 法人化（資産管理会社）コスト分析レポート (Incorporation Cost Breakdown)

調査日: 2025年12月27日
前提: 建物の所有・管理を主目的とする小規模な「資産管理法人」の設立

---

## 1. 設立時の初期費用 (Initial Setup Costs)

法人の種類「株式会社」にするか「合同会社」にするかで大きく異なります。
**小規模な家族経営であれば、初期費用が安く決算公告義務のない「合同会社」が推奨されます。**

### 株式会社 (KK) vs 合同会社 (GK)

| 項目 | 株式会社 (Kabushiki Kaisha) | 合同会社 (Godo Kaisha) |
| :--- | :--- | :--- |
| **定款認証手数料** | 約 50,000円 | **0円** (不要) |
| **定款印紙代** | 0円 (電子定款の場合) | 0円 (電子定款の場合) |
| **登録免許税** | 最低 150,000円 | **最低 60,000円** |
| **司法書士報酬** | 7〜11万円 | **5〜8万円** |
| **諸経費(印鑑等)** | 1〜2万円 | 1〜2万円 |
| **合計目安** | **約 24〜30万円** | **約 12〜16万円** |

## 2. 不動産（建物）の移転コスト (Property Transfer Costs)

建物を「個人から法人へ」売買して名義を変更する際にかかる税金です。
これは一度きりのコストですが、高額になりがちです。

**前提**: 土地は移転せず、建物のみ移転する想定。
**建物評価額の仮定**: 固定資産税評価額で **約1,000万円** と仮定して試算します。（※評価額は実際の納税通知書で確認が必要）

| 項目 | 税率 (店舗・非住宅) | 概算コスト (評価額1,000万の場合) | 備考 |
| :--- | :--- | :--- | :--- |
| **登録免許税** | 2.0% | **200,000円** | 登記変更時に支払い |
| **不動産取得税** | 4.0% | **400,000円** | 移転後、数ヶ月後に通知が来る |
| **印紙税** | (売買契約書による) | 10,000円 | 契約金額による |
| **司法書士報酬** | (登記代行) | 5〜10万円 | 所有権移転登記費用 |
| **合計目安** | | **約 66〜71万円** | **キャッシュアウトが発生します** |

※建物評価額がもっと低ければ（古い建物など）、この数値は下がります。

## 3. 毎年のランニングコスト (Annual Running Costs)

法人化すると、利益が出ていなくても発生するコストがあります。

### 税理士費用の変化

現在: **年10万円** (個人確定申告)
法人化後: **年25〜40万円** (程度にアップする予想)

| 項目 | 個人の場合 | 法人の場合 | 差額 |
| :--- | :--- | :--- | :--- |
| **決算・確定申告** | 10万円 | 15〜25万円 | +5〜15万 |
| **月額顧問料** | 0円 | 1〜2万円 × 12ヶ月 | +12〜24万 |
| **法人住民税(均等割)** | 0円 | **70,000円** (赤字でも必須) | +7万 |
| **年間合計** | **10万円** | **約 34〜56万円** | **約 +24〜46万円/年の増** |

### コストアップの解説

* **税理士報酬**: 法人の決算申告は個人の確定申告より複雑で、作成書類も多いため報酬相場が高くなります。
  * *対策*: 「訪問なし」「記帳は自分でやる（会計ソフト使用）」等の条件で、年20万円以下で受けてくれる格安税理士を探すことは可能です。
* **社会保険料**:
  * 貴方（役員）に給与を出す場合、法人は社会保険の加入義務が発生します（会社負担分＋本人負担分）。
  * これが結構重い（給与の約30%）ので、給与設定額には注意が必要です。

## 4. 収支シミュレーション (コスト対効果)

この「年間約30〜50万円のコスト増」を払ってでも法人化する意味があるか？

### 損益分岐点の考え方

コスト増（年50万円） ＜ **相続税圧縮効果** ＋ **所得分散効果**

1. **相続税圧縮**: 将来の相続税が数百万円減るなら、毎年の50万円は「保険料」として正当化できます。
2. **所得分散**: 貴方へ「月10万円」の給与を出せば、年間120万円の資産がお母様から貴方へ移転します。
    * 10年間で **1,200万円** の資産移転効果。
    * 10年間のコスト = 50万×10年 = **500万円**。
    * **差引 +700万円** 相当の資産防衛効果。

### 結論

* 単年の「手残り」だけ見れば、**法人化はコスト増（損）**になります。
* しかし、長期（10年以上）の**「資産承継・相続対策」**として見れば、十分に元が取れる投資になります。
